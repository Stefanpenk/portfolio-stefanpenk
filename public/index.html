Breaking Bad Charatcters

In this exercise our task were to get some data from The Breaking Bad API about characters appearing in the series.

    There is few extra things that I added to make it more interesting such as:
  • Like and Dislike counter. Everytime you hover over a character you can also choose between 2 icons (green and red) if you click one of them the counter above will change.
  • Search bar. If you are looking for a specific character you can type the full name or any number of chars.
  • Show Random. It will show you only one random character.
  • Show All. This is button to reset characters view.

Pokedex

In this exercise our task were to convert already prepared JSON file containing one pokemon and display pictures that were attached in this file as a link.

As a bonus I decided to create whole Pokedex which will not only display pictures but also pokemon name and basic stats.

But this Pokedex contains information about all Pokemons taken from PokAPI. To display Pokemon information you just need to write it's number in dedicated output.

What is more I added some CSS effects, so everytime you click any button there will be some animation.
